# pterodactyl-beammp-egg
Working BeamMP egg for pterodactyl


Want feature added? You can contact me! Or make a request in github!


How to install!

1. Open pterodactyl
2. go to admin panel
3. go to nests
4. upper right corner click add egg
5. select the .json file
6. select the folder
7. and import!

Enjoy the game!
